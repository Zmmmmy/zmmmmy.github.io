Project Reactor 实现

Project Reactor 简介

Project Reactor 是 Spring 官方的 Reactive Streams 实现，是 Spring WebFlux 的基础。

核心特点

  • 完全非阻塞：基于 JDK 8 函数式 API
  • 背压支持：完整实现 Reactive Streams 规范
  • 丰富的操作符：提供 100+ 操作符
  • 与 Spring 深度集成：WebFlux、Spring Data R2DBC 等
